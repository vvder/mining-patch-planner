# Mining Patch Planner

[Factorio Mod portal](https://mods.factorio.com/mod/mining-patch-planner)

## Development checks

### Lua runtime

This repository includes Lua source files for the Factorio mod runtime. To run local syntax checks, install a Lua runtime first.

Ubuntu/Debian:

```bash
sudo apt-get update
sudo apt-get install -y lua5.4
```

### Run syntax checks

```bash
./scripts/check-lua.sh
```
